## Zookeeper (from JetBrains Academy)

#### Program usage: 
Zookeeper Programm

#### Technologies used:
- *Python*

#### Usage 
```
python zookeeper.py
```

### Implementation:
(The greater-than symbol followed by a space (```> ```) in examples represents the user input. It's not part of the input.)

- Asking for the number of the desired habitat using the following phrase: ```Please enter the number of the habitat you would like to view: ```
- Using the input number as an index of existing habitats to print its content.
- Exit opportunity: inputting ```exit``` should end the program.
- When the program is done running, it prints: ```See you later!```
- Example: 

```
> python tictactoe.py
```
